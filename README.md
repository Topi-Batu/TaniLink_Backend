## Usage

Deploy Example
1. Clone Repo
  ```sh
  git clone https://github.com/Topi-Batu/TaniLink_Backend.git
  ```
2. Setting .env
  ```sh
  cp env.example .env
  nano .env
  ```
3. Deploy with Docker Compose
  ```sh
  docker compose up -d
  ```
3. Open
  ```sh
  Backend https://localhost:3505
  Machine Learning http://localhost:3504
  ```
